# Book Exchange

Цей проєкт — веб-додаток для обміну книгами між користувачами.

## Можливості

- Додавання та пошук книг
- Обмін книгами між користувачами
- Система реєстрації та авторизації
- Перегляд історії обмінів

## Технології

- Next.js, React, Tailwind

## Встановлення

1. Клонуйте репозиторій:
    ```bash
    git clone https://github.com/your-username/book-exchange.git
    ```
2. Встановіть залежності:
    ```bash
    cd book-exchange
    npm install
    ```
3. Запустіть сервер:
    ```bash
    npm run dev
    ```
